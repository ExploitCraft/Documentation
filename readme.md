# terminal-page

A hacking-themed terminal portfolio page. Built with pure HTML, CSS, and JS — no frameworks, no dependencies.

## Features

- Matrix rain background
- CRT scanline + flicker effect
- Animated skill bars
- Interactive fake terminal (try `help`)
- Custom blinking cursor

## Usage

Just open `index.html` in a browser. No install needed.

## Commands

Type into the input bar at the bottom:

`whoami` `skills` `projects` `contact` `ls` `pwd` `date` `clear`

## Deploy

Works on GitHub Pages, Cloudflare Pages, or anywhere that serves static HTML.
